## Welcome to VedaTrace

VedaTrace is a universal logging solution designed for modern applications. It provides a simple, type-safe, and lightweight SDK for capturing and sending logs from your applications to the VedaTrace platform.

### Key Features

- **Universal:** Works in Node.js, Cloudflare Workers, browsers, and more.
- **Type-safe:** Full TypeScript support with autocomplete.
- **Lightweight:** Minimal performance overhead.
- **Developer-friendly:** Easy to use API and framework integrations.

### Getting Started

To get started, choose the SDK for your programming language from the sidebar.
