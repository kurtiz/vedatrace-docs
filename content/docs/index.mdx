## Observability without the Overkill

VedaTrace is a minimalist, edge-powered logging and observability platform designed to provide developers with actionable signals for fixing bugs without the complexity of traditional dashboards.

It is built on Cloudflare Workers for &lt;50ms latency and zero-cold-start ingestion, and provides AI-powered debugging to analyze stack traces and suggest fixes in plain English.

### Key Features

- **Zero-Config PII Scrubbing**: Automatically redacts passwords, secrets, and keys by default.
- **Soft-Cap Policy**: Prevents "denial of wallet" by keeping logs during spikes instead of dropping them or charging surprise overage fees.
- **Simple Search**: No complex query languages; uses a Google-like search interface.
- **No Vendor Lock-in**: Allows users to export their data at any time.
