## Installation

```bash
go get github.com/KOFI-GYIMAH/vedatrace-go
```

**Requirements:** Go 1.21+. Zero external dependencies.

## Basic Usage

```go
package main

import (
    "log"
    vedatrace "github.com/KOFI-GYIMAH/vedatrace-go"
)

func main() {
    logger, err := vedatrace.New(vedatrace.Config{
        APIKey:  "your-api-key",
        Service: "my-service",
    })
    if err != nil {
        log.Fatal(err)
    }
    defer logger.Stop()

    logger.Info("Hello world")
    logger.Error("Something went wrong", err)
}
```

## Development Mode

For local development, use `NewDev` to log to the console without sending data to VedaTrace. No API key is required.

```go
import vedatrace "github.com/KOFI-GYIMAH/vedatrace-go"

// Console-only logging for development
logger := vedatrace.NewDev("my-service")

logger.Info("This is a test log")
```
