Need help? Found a bug? Have an idea? Email us at **support@vedatrace.dev**.

To help us resolve your issue quickly, please use the format below.

## Report Template

```
Category:   (Dashboard / SDK / Docs)
Type:       (Bug Report / Feature Request / Feedback / Question)
Subject:    Brief summary of the issue or suggestion

Description:
A clear description of what's happening, what you'd like to see, or what you're wondering about.

Environment:
- Dashboard: Browser name + version, attach screenshot or screen recording if applicable
- SDK: Language + version, runtime (Node.js / Cloudflare Workers / Bun / Deno / Browser), SDK version, minimal reproduction code if possible
- Docs: Page URL, describe what's incorrect, missing, or unclear

Steps to reproduce (for bugs):
1.
2.
3.

Expected behavior:
What should happen.

Actual behavior:
What actually happens instead.
```

## What to Include

- **Dashboard issues** — browser, OS, screenshot or screen recording of the problem
- **SDK issues** — language, SDK version, runtime environment, and a minimal code reproduction
- **Docs issues** — the page URL and what's wrong or missing
