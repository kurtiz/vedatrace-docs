---
title: 'Managing API Keys'
description: Learn how to create and manage API keys for authenticating your applications.
---

API keys authenticate your applications when sending logs to VedaTrace. Each project can have multiple API keys.

## Creating an API Key

1. Navigate to **API Keys** in the sidebar
2. Click **Generate Key**
3. Enter a name for your key (e.g., "Production App", "Development")
4. Click **Generate**

## Important: Save Your Key

⚠️ **Your API key will only be shown once!**

After creating a key, you'll see a modal with your key. Copy it immediately and store it securely - you won't be able to see it again.

## Using Your API Key

Pass the API key to the VedaTrace SDK in your code:

```typescript
import { vedatrace } from 'vedatrace'

const logger = vedatrace({
  apiKey: 'veda_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
  service: 'my-service'
})
```

## Revoking API Keys

If an API key is compromised or no longer needed, you can revoke it:

1. Go to **API Keys**
2. Find the key you want to revoke
3. Click the trash icon
4. Confirm the revocation

> ⚠️ **Warning:** Revoking an API key will immediately block any applications using that key from sending logs.

## Key Security Tips

- Never commit API keys to version control
- Use different keys for different environments
- Revoke keys that are no longer in use
- Rotate keys periodically for better security
