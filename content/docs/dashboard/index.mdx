The VedaTrace dashboard is your command center for managing logs, API keys, and alert notifications. This guide will walk you through everything you need to get started.

## Quick Start

1. **Create a Project** - Organize your logs by project
2. **Get Your API Key** - authenticate your application
3. **Send Logs** - Start logging from your code
4. **Set Up Alerts** - Get notified when issues occur

## Navigation Sidebar

The sidebar on the left provides quick access to:

- **Dashboard** - Overview of your logs and usage
- **Projects** - Manage your projects
- **API Keys** - Manage authentication keys
- **Logs** - Search and view all logs
- **Settings** - Account, billing, and alert configuration
